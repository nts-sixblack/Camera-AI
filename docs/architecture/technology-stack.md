# Technology Stack

## Core Technologies

| Layer | Technology | Purpose |
|-------|------------|---------|
| **Language** | Swift 6.x | Primary development language |
| **UI** | SwiftUI | Declarative UI framework |
| **Camera** | AVFoundation | Camera capture and control |
| **Processing** | Metal | Real-time image processing |
| **Storage** | PhotoKit | Photo library integration |

## Minimum Requirements

- **iOS Version**: 17.0+
- **Device**: iPhone only (no iPad support)
- **Xcode**: 15.0+
