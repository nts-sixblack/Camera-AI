# Security & Privacy

## Permissions Required

| Permission | Usage | Required |
|------------|-------|----------|
| Camera | Photo/video capture | Yes |
| Photo Library | Save captures | Yes |
| Microphone | Video recording | Optional |
| Location | Geotagging | Optional |

## Data Policy

- **Offline-first:** No internet required
- **No data collection:** No analytics or telemetry
- **No account system:** No user registration
- **Local storage only:** All data stays on device
