# Testing Architecture

## Test Strategy

| Test Type | Location | Purpose |
|-----------|----------|---------|
| Unit Tests | `CameraTests/` | Component isolation testing |
| UI Tests | `CameraUITests/` | End-to-end flow testing |
| Performance Tests | `CameraPerformanceTests.swift` | Benchmark verification |

## Test Coverage Focus

- Permission manager state transitions
- CameraEngine lifecycle
- View state rendering
- Launch performance
