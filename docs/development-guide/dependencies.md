# Dependencies

## System Frameworks Only

This project uses only Apple system frameworks:

| Framework | Purpose |
|-----------|---------|
| SwiftUI | User interface |
| AVFoundation | Camera capture |
| PhotoKit | Photo library |
| CoreLocation | Geotagging (optional) |
| Metal | Image processing |

## No Third-Party Dependencies

- No CocoaPods
- No Swift Package Manager dependencies
- No Carthage

This keeps the project lightweight and reduces potential compatibility issues.
