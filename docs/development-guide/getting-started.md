# Getting Started

## 1. Clone and Open Project

```bash
# Navigate to project directory
cd /path/to/Camera

# Open in Xcode
open Camera.xcodeproj
```

## 2. Configure Signing

1. Open `Camera.xcodeproj` in Xcode
2. Select the "Camera" target
3. Go to **Signing & Capabilities** tab
4. Select your development team
5. Ensure "Automatically manage signing" is checked

## 3. Select Device

1. Connect your iPhone via USB or Wi-Fi
2. Select your device from the scheme dropdown (not a Simulator)
3. Trust your computer on the device if prompted
