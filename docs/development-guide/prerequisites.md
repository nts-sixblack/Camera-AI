# Prerequisites

## Required Software

| Software | Version | Notes |
|----------|---------|-------|
| macOS | 14.0+ (Sonoma) | For Xcode 15 support |
| Xcode | 15.0+ | iOS development IDE |
| iOS Device | iPhone with iOS 17+ | Physical device required for camera testing |

## Hardware Requirements

- **Mac**: Apple Silicon (M1+) or Intel Mac with 16GB+ RAM
- **iPhone**: Physical device required (camera APIs don't work in Simulator)
- **Apple Developer Account**: Required for device deployment
