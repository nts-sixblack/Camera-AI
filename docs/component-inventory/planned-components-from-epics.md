# Planned Components (from Epics)

Based on the PRD and Epics, the following components are planned:

## Epic 2: Manual Controls

| Component | Purpose |
|-----------|---------|
| `ISODialView` | ISO adjustment control |
| `ShutterSpeedDialView` | Shutter speed control |
| `FocusDialView` | Manual focus control |
| `WhiteBalancePickerView` | WB presets and Kelvin |
| `ProModeToggleView` | Auto/Pro mode switch |

## Epic 3: Visual Aids

| Component | Purpose |
|-----------|---------|
| `HistogramOverlayView` | Real-time RGB histogram |
| `FocusPeakingOverlayView` | Focus peaking visualization |
| `AELockIndicatorView` | Auto-exposure lock status |
| `AFLockIndicatorView` | Auto-focus lock status |

## Epic 4: Capture & Assets

| Component | Purpose |
|-----------|---------|
| `FormatSelectorView` | RAW/HEIC/JPG selection |
| `GeotaggingToggleView` | Location metadata control |
| `SaveProgressView` | Background save indicator |
| `StorageWarningView` | Low storage notification |
